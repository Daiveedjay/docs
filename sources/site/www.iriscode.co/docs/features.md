# Features

Everything Iris can do, from a single file to your entire workspace.

[File Analysis\\ \\ Per-file health scores, complexity, TypeScript metrics, code smells, and warnings - updated on every save.](https://www.iriscode.co/docs/features/file-analysis) [Language Support\\ \\ TypeScript, JavaScript, Go, and Python - with framework-aware parsing and configurable rules per language.](https://www.iriscode.co/docs/features/language-support) [Workspace Analysis\\ \\ Aggregate metrics across every file in your project, with folder breakdowns and exportable reports.](https://www.iriscode.co/docs/features/workspace-analysis) [Editor Integration\\ \\ CodeLens overlays, status bar health indicators, and a detached analysis panel built into VS Code.](https://www.iriscode.co/docs/features/editor-integration)
