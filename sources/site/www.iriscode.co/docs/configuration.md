# Configuration

Tune thresholds and behaviour across your team.

[VS Code Settings\\ \\ Open Ctrl+, and search Iris to configure thresholds, toggles, and display options per machine.](https://www.iriscode.co/docs/configuration/vscode-settings) [.irisconfig.json\\ \\ Drop a .irisconfig.json at your project root so every developer runs Iris with the same thresholds. Includes severity overrides.](https://www.iriscode.co/docs/configuration/irisconfig) [Config Studio\\ \\ New\\ \\ Visual editor in your account dashboard to build, preview, and sync .irisconfig.json to VS Code in one click.](https://www.iriscode.co/docs/configuration/config-studio) [Scoring Weights\\ \\ New\\ \\ Control how much each finding type dents the health score. Amplify secrets, zero out noise, or match your team's priorities.](https://www.iriscode.co/docs/configuration/scoring-weights)
